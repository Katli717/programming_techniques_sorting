\doxysubsection{Detailed Description}
Заголовочный файл для работы с данными о солдатах. 

Этот файл содержит определения структур для хранения данных о солдатах, а также функции для сортировки и работы с этими данными.

Структуры\+:
\begin{DoxyItemize}
\item \doxylink{structFIO}{FIO}\+: хранит фамилию, имя и отчество солдата.
\item \doxylink{structSoldier}{Soldier}\+: хранит данные о солдате, включая его звание, ФИО, номер взвода и возраст.
\end{DoxyItemize}

Операторы сравнения для структур\+:
\begin{DoxyItemize}
\item Операторы сравнения (==, !=, \texorpdfstring{$<$}{<}, \texorpdfstring{$>$}{>}, \texorpdfstring{$<$}{<}=, \texorpdfstring{$>$}{>}=) реализованы для структур \doxylink{structFIO}{FIO} и \doxylink{structSoldier}{Soldier}, что позволяет удобно сравнивать объекты этих структур.
\end{DoxyItemize}

Функции\+:
\begin{DoxyItemize}
\item selection\+Sort\+: сортирует вектор солдат с использованием алгоритма сортировки выбором.
\item insertion\+Sort\+: сортирует вектор солдат с использованием алгоритма сортировки вставками.
\item heapify\+: вспомогательная функция для пирамидальной сортировки.
\item heap\+Sort\+: сортирует вектор солдат с использованием алгоритма пирамидальной сортировки.
\item read\+Soldiers\+From\+File\+: читает данные о солдатах из указанного файла и возвращает вектор объектов \doxylink{structSoldier}{Soldier}.
\item write\+Soldiers\+To\+File\+: записывает данные о солдатах в указанный файл. 
\end{DoxyItemize}
