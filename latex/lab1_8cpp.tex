\doxysubsection{Detailed Description}
Реализация функций для работы с данными о солдатах. 

Этот файл содержит реализацию методов для структур \doxylink{structFIO}{FIO} и \doxylink{structSoldier}{Soldier}, а также функций для сортировки и работы с данными солдат.

Реализация методов\+:
\begin{DoxyItemize}
\item FIO\+::\+FIO\+: Конструктор для инициализации объекта \doxylink{structFIO}{FIO} с заданными фамилией, именем и отчеством.
\item Операторы сравнения для \doxylink{structFIO}{FIO}\+: реализованы операторы ==, !=, \texorpdfstring{$<$}{<}, \texorpdfstring{$>$}{>}, \texorpdfstring{$<$}{<}=, \texorpdfstring{$>$}{>}= для сравнения объектов \doxylink{structFIO}{FIO}.
\item Soldier\+::\+Soldier\+: Конструктор для инициализации объекта \doxylink{structSoldier}{Soldier} с заданным званием, ФИО, номером взвода и возрастом.
\item Операторы сравнения для \doxylink{structSoldier}{Soldier}\+: реализованы операторы ==, !=, \texorpdfstring{$<$}{<}, \texorpdfstring{$>$}{>}, \texorpdfstring{$<$}{<}=, \texorpdfstring{$>$}{>}= для сравнения объектов \doxylink{structSoldier}{Soldier}.
\end{DoxyItemize}

Функции сортировки\+:
\begin{DoxyItemize}
\item selection\+Sort\+: Реализует алгоритм сортировки выбором для сортировки вектора солдат.
\item insertion\+Sort\+: Реализует алгоритм сортировки вставками для сортировки вектора солдат.
\item heapify\+: Вспомогательная функция для поддержания свойств кучи в пирамидальной сортировке.
\item heap\+Sort\+: Реализует алгоритм пирамидальной сортировки для сортировки вектора солдат.
\end{DoxyItemize}

Функции для работы с файлами\+:
\begin{DoxyItemize}
\item read\+Soldiers\+From\+File\+: Читает данные о солдатах из указанного файла и возвращает вектор объектов \doxylink{structSoldier}{Soldier}. Ожидает, что каждая строка файла содержит информацию о солдате в определенном формате.
\item write\+Soldiers\+To\+File\+: Записывает данные о солдатах в указанный файл в текстовом формате.
\end{DoxyItemize}

Примечание\+: Все функции обработки и сортировки работают с вектором объектов \doxylink{structSoldier}{Soldier}, который содержит информацию о солдатах и их характеристиках.
